  this demo provides a description of how to use the two usb otg host peripheral.
  
  when an usb device is attached to the otghs host port, the device is enumerated and
  checked whether it msc device.

  when an usb device is attached to the otgfs host port, the device is enumerated and
  checked whether it can support hid device or not,the demo just support keyboard
  and mouse device.

  for more detailed information, please refer to the application note document AN0094.
