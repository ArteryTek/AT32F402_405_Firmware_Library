  this demo is based on the at-start board. qspi flash programming algorithms
  is showed in this demo.  the target flash is esmt32m. 
  the following pin need tobe select by Configuration Wizard:
  - QSPI1 CS pin	: PA2/PB6/PB9/PB10/PC11
  - QSPI1 SCK pin	: PB1/PB2/PB4
  - QSPI1 IO0 pin	: PA6/PB0/PB5/PB11/PC9/PF6
  - QSPI1 IO1 pin	: PA7/PB7/PB10/PC10
  - QSPI1 IO2 pin	: PA6/PA15/PC4/PC8
  - QSPI1 IO3 pin	: PA1/PB3/PC5
